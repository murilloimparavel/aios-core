# Raw - Material Não Processado

## 🗄️ Propósito

Esta pasta é o "inbox" para todo material coletado que ainda não foi categorizado ou processado.

---

## 📥 Workflow

1. **Download** → Todo material vai primeiro para `raw/`
2. **Review** → Verificar qualidade e relevância
3. **Categorize** → Mover para pasta apropriada
4. **Process** → Extrair insights e integrar

---

## 📋 Guidelines

### O que vai aqui:
- ✅ Downloads iniciais antes de categorização
- ✅ Material que precisa ser reviewed
- ✅ Arquivos grandes temporários
- ✅ Bulk downloads de scraping
- ✅ Backups temporários

### O que NÃO vai aqui:
- ❌ Material já categorizado
- ❌ Material já processado
- ❌ Arquivos permanentes

---

## 🔧 Organização Sugerida

```
raw/
  ├── YYYYMMDD_batch_description/
  │   ├── files...
  │   └── manifest.txt
  └── temp/
      └── processing...
```

### Batch Naming
- Use data + descrição: `20251010_youtube_videos_batch1/`
- Inclua manifest.txt listando conteúdo
- Mantenha organizado por data de coleta

---

## 🧹 Limpeza

**Política:** Limpar a cada 30 dias
- Mover processado para categorias corretas
- Deletar material rejeitado
- Arquivar backups se necessário
