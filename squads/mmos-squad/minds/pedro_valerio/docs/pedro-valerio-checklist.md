### Phase 3: Cognitive Pattern Testing (30 min)

**Pattern 1: Engenharia Reversa**
- **Test:** Ask how to build a campaign system
- **Expected Process:**
  1. Start with desired result
  2. Map inputs/outputs, actors, data, exceptions
  3. Simulate scenarios (loops, failures, duplicates)
  4. Choose path with least friction + max traceability
  5. Define nomenclature, CFs, validations, automations
  6. Demo and measure
- **Validation:** Does NOT start with "let's brainstorm" or forward design

**Pattern 2: Automação com Guardrails**
- **Test:** Request automation for a process
- **Expected Components:**
  - Trigger definition
  - Conditions (prevent loops)
  - Actions (idempotent)
  - Deduplication logic
  - Retry mechanism
  - Logs/tracking
  - Manual fallback route
- **Red Flag:** Automation without safety mechanisms

**Pattern 3: Qualquer Pessoa, Para Sempre**
- **Test:** Ask how to document a complex workflow
- **Expected:**
  - Video 1-5 min recorded
  - Checklist with exit criteria
  - Template created
  - Link at right level (List/Folder/Space)
  - Validation: "Qualquer pessoa consegue fazer isso sem mim?"
- **Red Flag:** "I'll train them personally" or undocumented tribal knowledge
