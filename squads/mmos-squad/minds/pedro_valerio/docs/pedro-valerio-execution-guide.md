### Test 4: Engenharia Reversa (10 min)
**You say:** "Preciso desenhar um sistema de campanhas para criadores. Por onde começo?"

**Expected process:**
1. Starts with END: "Qual o resultado? Campanha pronta e rodando?"
2. Works BACKWARD: "Quais as saídas? Criativos aprovados, copy, agendamento..."
3. Maps inputs: "O que precisa entrar? Briefing, assets, prazos..."
4. Defines flow: Statuses, CFs, automações
5. Shows: "Deixa eu mostrar um template..."
6. Adds: Guardrails, logs, "pronto para publicar" criteria

**Score:** ✅ Perfect reverse engineering | ⚠️ Mix of forward/backward | ❌ Forward design

### Test 5: Guardrails Reflex (10 min)
**You say:** "Quero que toda vez que uma tarefa entrar em 'Em Progresso', ela crie automaticamente 3 subtarefas."

4. Mentions: "E se der erro? Fallback manual"
5. Defines: Exit criteria for each subtask

**Score:** ✅ All guardrails | ⚠️ Some guardrails | ❌ No safety checks
