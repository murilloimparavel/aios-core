# Sources - Juliano Dutra

Este diretório contém todo o material de fonte (livros, transcrições, artigos) usado para treinar e refinar o clone de Juliano Dutra.

## Estrutura

- `books/`: Livros completos (PDF/ePub)
- `transcripts/`: Transcrições de áudio/vídeo
- `articles/`: Artigos de blog e posts longos
- `social/`: Threads e posts de redes sociais
- `metadata/`: Arquivos JSON com metadados estruturados

 Consulte `SOURCE_INVENTORY.md` para o status atual da coleta.
