# Paul Graham Essays Database

## Visão Geral

Esta pasta contém uma estrutura organizada de referências para todos os 232 essays de Paul Graham, capturados de forma respeitosa para análise acadêmica e estudo de padrões cognitivos.

## Estrutura

### 📁 `essay_references/`
- **232 arquivos de referência** em formato Markdown
- Cada arquivo contém metadados, links originais e templates para análise
- **000_INDEX.md** - Índice completo com todos os essays

### 📄 `scrape_essays.py`
- Script Python usado para capturar as referências
- Respeita direitos autorais (apenas títulos e links)
- Pode ser executado novamente para atualizações
